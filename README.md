# Matrix-Appender
Turns a list of n<sup>2</sup>, n x n matrices into an n<sup>2</sup> x n<sup>2</sup> matrix. Keeping the form of each n x n matrix

Example:

a = [2 0] b = [5 5] c = [6 6]  d = [1 1]
    [4 4]     [6 7]     [9 4]     [5 5]
    

Array List = [a, b, c, d]

Final Array = [2 0 5 5]
              [4 4 6 7]
              [6 6 1 1]
              [9 4 5 5]
