# Matrix-Appender
Turns a list of n<sup>2</sup>, n x n matrices into an n<sup>2</sup> x n<sup>2</sup> matrix. Keeping the form of each n x n matrix

Example:

<pre>
a = [2 0]  
    [4 4]                

b = [5 5]<br/>
    [6 7]

c = [6 6]<br/>
    [9 4]

d = [1 1]<br/>
    [5 5]

Array List = [a, b, c, d]

Final Array = [2 0 5 5]<br/>
              [4 4 6 7]<br/>
              [6 6 1 1]<br/>
              [9 4 5 5]<br/>
</pre>
